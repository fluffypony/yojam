Yojam Extension Settings

How it works

Click the toolbar button or right-click any link and choose "Open Link in Yojam" to route it through Yojam's browser picker.

You can also use the keyboard shortcut Alt+Shift+Y to send the current tab's URL to Yojam.

Native messaging

For the smoothest experience, make sure the Yojam native messaging host is installed. Open Yojam's preferences and go to Integrations to check the status and reinstall if needed.
